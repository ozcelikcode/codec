# Product Context

## Why this exists
Users need reliable, high-fidelity codec conversion and resizing for modern web delivery and professional workflows.

## User-facing workflow
1. Upload one or multiple image files.
2. Choose target codec and optional dimensions.
3. Start conversion and poll status.
4. Download resulting file(s) when complete.

## UX requirements reflected in backend
- Batch support.
- Async lifecycle with status polling.
- Clear metadata and summary reporting.
- Privacy-first retention and auto-cleanup.
