# Project Brief

## Goal
Build a production-grade backend for image upload, conversion, resizing, optimization, and download with strong maintainability and extensibility.

## Scope Baseline
- REST API with upload, convert, status, and download.
- Async processing lifecycle with polling.
- Strong validation, structured errors, and predictable contracts.
- Local ephemeral storage in v1 with cleanup.

## Quality Bar
- Senior-level modular architecture.
- Clean separation of concerns.
- Deterministic processing behavior.
- Future-ready for queue workers and cloud storage.
